Simple Google Map

Description

Using $("#map").googleMap() to load the map.This map needs a jquery library jquery.googlemap.js,which is designed for google map.

Code Snippet

JS

$("#map").googleMap() to call the function.

 
        $(function () {
        $("#map").googleMap({
            zoom: 16,
            coords: [42.340401, -71.088806], 
            type: "ROADMAP" 
        });
    })
